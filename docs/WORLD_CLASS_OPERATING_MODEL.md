# World-Class Team Operating Model

## Purpose
Create execution consistency independent of any single contributor.

## Core Roles

1. Program Lead: roadmap and cross-stream alignment.
2. Evidence Lead: KPI integrity and validation quality.
3. Engineering Lead: reliability, security, and delivery.
4. Partnerships Lead: institutional onboarding and retention.
5. Communications Lead: transparent reporting discipline.

## Operating Cadence

1. Weekly
- Delivery standup with blockers and risk updates.

2. Monthly
- Workstream scorecard review.
- Governance and ethics review.

3. Quarterly
- Strategy checkpoint.
- Public progress publication.

## Decision Framework

1. Reversible decisions: fast owner-level execution.
2. Irreversible decisions: cross-functional review.
3. Public impact claims: evidence sign-off required.

## Success Rule

Execution quality is measured by outcome reliability, not activity volume.
