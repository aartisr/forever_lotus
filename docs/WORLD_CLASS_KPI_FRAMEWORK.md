# World-Class KPI Framework

## Objective
Measure real-world impact with evidence quality high enough for external scrutiny.

## KPI Categories

1. Impact KPIs
- Beneficiaries reached.
- Verified reduction in target harms.
- Improvement in dignity and agency indices.

2. Evidence KPIs
- Number of independent evaluations completed.
- Replication success rate across regions.
- Peer-reviewed publications and citations.

3. Reliability KPIs
- Uptime against SLO target.
- P95 latency by critical workflow.
- Incident count, MTTR, and recurrence rate.

4. Trust and Governance KPIs
- Ethics incidents and closure time.
- Audit findings and remediation SLA.
- Stakeholder trust score trend.

5. Adoption KPIs
- Active institutional partners.
- Countries/regions with active deployment.
- Percentage of local-lead implementations.

## Measurement Discipline

1. Every KPI must define:
- Name.
- Formula.
- Source system.
- Collection frequency.
- Owner.
- Data quality checks.

2. All public KPI reporting must include:
- Baseline value.
- Current value.
- Confidence/uncertainty notes.
- Known limitations.

## Baseline Setup Checklist

1. Define KPI dictionary and owners.
2. Set collection and validation pipeline.
3. Capture baseline before intervention.
4. Freeze baseline snapshot with timestamp.
5. Publish versioned methodology.
